# Zion Streaming

The official website for Zion Streaming / Zion Media — a pro–Latter-day Saint media studio dedicated to bold truth, sacred history, prophecy, and the gathering of Israel.

**Live at:** [zionstreaming.com](https://www.zionstreaming.com)

## Stack

Static HTML, CSS, and SVG. No frameworks, no build step. Hosted on Vercel, with GitHub as source-of-truth and GoDaddy DNS pointing the domain.

## Structure

```
/
├── index.html                  Home
├── about/index.html            About / Team
├── shows/index.html            Podcasts hub
├── gatherings/index.html       Upcoming + past gatherings
├── resources/index.html        Study manuals + archive
├── contact/index.html          Contact info
├── privacy-policy.html         Privacy
├── terms-conditions.html       Terms
├── 404.html                    Not found
├── styles.css                  All styles (single file)
└── images/                     Logo + favicon
```

## Editing

Any text edit can be done directly on GitHub by clicking the pencil icon on the file. Vercel auto-deploys within 60 seconds of every commit.

## Brand

- **Display type:** Fraunces (Google Fonts)
- **Body type:** DM Sans (Google Fonts)
- **Color:** Strict black (`#0a0a0a`) + off-white paper (`#fafaf7`)
